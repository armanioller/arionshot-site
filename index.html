Recursos

Feito para quem precisa explicar algo visualmente, sem abrir um editor pesado.

01

Overlay ajustável

Abra pela bandeja ou atalho, selecione uma área, ajuste pelos cantos, mova a seleção e veja as dimensões em tempo real.

02

Anote antes de enviar

Adicione setas, texto, retângulos, círculos e tarjas para transformar um print em instrução clara.

03

Google Drive

Conecte sua conta, salve na pasta Arion Prints e receba o link compartilhável automaticamente.

04

Envio direto

Compartilhe uma captura diretamente com outro usuário do ArionShot, ideal para testes em rede local.

05

Feedback integrado

O usuário pode enviar relato, print e dados técnicos pelo próprio app. O report chega em uma área privada para triagem.

Como funciona

Do print ao link em poucos cliques.

  1. CaptureUse o atalho ou dê dois cliques no ícone da bandeja.
  2. AjusteRedimensione a seleção, mova a área e acompanhe as dimensões.
  3. DestaqueUse anotações para apontar exatamente o que importa.
  4. CompartilheSalve, copie, envie para o Drive ou mande direto para outro ArionShot.
